# The Executive | Barbearia Premium

Site institucional desenvolvido para demonstração de uma barbearia de alto padrão.

## 🚀 Tecnologias
- HTML5
- CSS3 (Custom Properties, Glassmorphism, Responsive Design)
- JavaScript (ScrollReveal, Smooth Scroll, Form Handling)
- Google Fonts & Font Awesome

## 🎨 Design
O design foi construído com foco em uma estética premium, utilizando:
- **Dark Mode** elegante com tons de dourado/âmbar.
- **Glassmorphism** nos cards de serviços e formulários.
- **Animações de entrada** suaves para uma navegação fluida.
- **Mobile First**: Totalmente responsivo para smartphones e tablets.

## 📄 Como visualizar
1. Clone o repositório.
2. Abra o arquivo `index.html` em seu navegador.
